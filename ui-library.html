Foundation

Typography

All text styles used across index.html and case-study.html. One variable font: Mona Sans across all roles — weight 200–900, width 75–125%, with optical sizing. Headings use weight 700–800 at 87.5% width (semi-condensed). Body & UI use weight 400–500 at normal width. Base: 16px.

Designing for real people

Explorations & Iterations

Hero image — Big

Full-bleed case study opener. Single image, fixed 1025 × 768px, cursor zooms into lightbox on click. Class: .cs-hero-image

1025 × 768 px 4 : 3 object-fit: cover click → lightbox
HTML 
<!-- Wrap in .container for page-level padding -->
<div class="cs-hero-image">
  <div class="container">
    <img
      src="image.jpg"
      alt="Description"
      loading="lazy"
      data-lightbox-solo="image.jpg"
      data-lightbox-alt="Description"
      style="cursor: zoom-in;"
    >
  </div>
</div>

Component 3 of 7

Image Grid

Two-column image grid for side-by-side visual comparisons. Each cell locks to a 4:3 ratio via aspect-ratio. Each image opens in the lightbox on click. Class: .cs-image-grid

2-column grid each cell: 4 : 3 gap: 1rem click → lightbox
HTML 
<div class="cs-image-grid">
  <img
    src="image-a.jpg"
    alt="Caption A"
    loading="lazy"
    data-lightbox="image-a.jpg"
  >
  <img
    src="image-b.jpg"
    alt="Caption B"
    loading="lazy"
    data-lightbox="image-b.jpg"
  >
</div>

Component 4 of 7

Carousel — Standard

Inline carousel at exactly 680 × 480px, centered on the prose column's midpoint. Used in the Research and Design Process sections. Swipeable on touch; dot + arrow controls. Class: .cs-carousel

slides: 680 × 480 px 17 : 12 width: 100% of container swipe + dot + arrow controls
HTML 
<div class="cs-carousel" aria-label="Label">
  <div class="cs-carousel__track-wrap">
    <div class="cs-carousel__track">
      <div class="cs-carousel__slide">
        <img src="slide-1.jpg" alt="Slide 1" loading="lazy">
      </div>
      <div class="cs-carousel__slide">
        <img src="slide-2.jpg" alt="Slide 2" loading="lazy">
      </div>
      <!-- repeat .cs-carousel__slide for each additional slide -->
    </div>
  </div>
  <div class="cs-carousel__controls">
    <div class="cs-carousel__dots">
      <button class="cs-carousel__dot active" aria-label="Slide 1"></button>
      <button class="cs-carousel__dot" aria-label="Slide 2"></button>
      <!-- one dot per slide -->
    </div>
    <div class="cs-carousel__arrows">
      <button class="cs-carousel__btn" data-carousel-prev aria-label="Previous">&larr;</button>
      <button class="cs-carousel__btn" data-carousel-next aria-label="Next">&rarr;</button>
    </div>
  </div>
</div>
